**Description:**\
\
A Command Line Interface (CLI) account management system built in
Python.\
\
The program allows users to:

-   Create an account

-   Login to an existing account

-   Delete an account\
    \
    \
    **How to Run:**\
    \
    1. Ensure Python is installed (Python 3 is recommended).\
    2. Ensure common_password.json exists in the same directory.\
    3. Run the script:\
    \
    \`\`\`bash\
    python3 Password_Manager.py\
    \`\`\`\
